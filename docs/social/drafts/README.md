# Social drafts

This is where the social media agent saves each week's proposed content for
review, before anything is published.

- One file per week, named by date, for example `2026-07-06.md`.
- Each file holds: the week's plan, every carousel written slide by slide, the
  captions, hashtags, any image direction, and the suggested posting schedule.
- Nothing in here is published until the owner approves it.

See `../AGENT-BRIEF.md` for how the agent works and the rules it follows.
