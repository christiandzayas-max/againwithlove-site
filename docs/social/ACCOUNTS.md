# Meta Accounts Setup (Facebook + Instagram)

Goal: claim the brand on Facebook and Instagram now, set them up cleanly, and
connect them so an agent can help run them (with a human approving what posts).

## Handles
Try to get the same handle on both platforms for consistency. Grab it now even
before we post, so no one else takes it.
- First choice: **@againwithlove**
- Fallbacks: @again.with.love, @againwithlove.care, @againwithloveapp

## Do these steps (in order)
1. **Facebook Page.** From a personal Facebook account, create a **Page** named
   **Again, With Love**. Category: choose something like "Health & Wellness
   Website" or "Community." Avoid medical-provider categories (Doctor, Medical
   Company) so we are never mistaken for a healthcare provider.
2. **Instagram.** Create an Instagram account with the chosen handle, then in
   Settings switch it to a **Professional account** (Business or Creator).
3. **Connect them.** In Meta Business Suite / Accounts Center, link the Instagram
   account to the Facebook Page. This is what later allows scheduling and API
   posting.
4. **Business Portfolio.** Create a Meta **Business Portfolio** (formerly Business
   Manager) to hold the Page, the Instagram account, and any future ad account.
   This is also where an agent's access would be granted later.
5. **Profile art.** Use the brand assets from the site repo `public/` folder:
   the gold infinity-heart (`logo-icon.png`) as the profile photo. A cover image
   can come later.
6. **Website link.** Point both profiles to **https://againwithlove.com**.
